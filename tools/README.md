# Ferramentas auxiliares

Ferramentas e artefatos de suporte ao desenvolvimento e deploy.

## Conteúdo

| Pasta/Arquivo | Descrição |
|---------------|-----------|
| `ssh/` | Chaves SSH para acesso à VPS Oracle (não versionadas) |

## Segurança

Chaves privadas (`*.key`, `*.pem`) estão no `.gitignore` e **nunca** devem ser commitadas.
Apenas a chave pública (`.pub`) pode ser compartilhada com segurança.
