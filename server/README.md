# Diretório reservado para referência à instalação Steam.

A instalação oficial do DayZ Server é uma dependência externa e não faz parte deste repositório.

Caminho padrão (Windows):

```
C:\Program Files (x86)\Steam\steamapps\common\DayZServer
```

Este diretório `server/` pode ser usado no futuro para symlinks, scripts de ponte ou documentação de caminhos locais.
