# Architecture Decision Records (ADR)

Este diretório contém **Architecture Decision Records** — documentos que registram decisões técnicas importantes do projeto.

## Formato

Use o template abaixo para novos ADRs. Numere sequencialmente: `0001-titulo-curto.md`, `0002-outro-titulo.md`, etc.

## Template

```markdown
# ADR-NNNN: Título da decisão

**Status:** Proposto | Aceito | Depreciado | Substituído  
**Data:** YYYY-MM-DD  
**Autores:** Nome

## Contexto

Qual problema ou necessidade motivou esta decisão?

## Decisão

O que foi decidido?

## Consequências

### Positivas
- ...

### Negativas
- ...

## Alternativas consideradas

- Alternativa A — motivo de rejeição
- Alternativa B — motivo de rejeição
```
