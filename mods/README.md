# Mods

Referências e manifests de mods do servidor. **Binários de mods não são versionados** — eles vêm da Steam Workshop.

## Mods ativos

| Mod | Pasta | Workshop ID |
|-----|-------|-------------|
| Community Framework (CF) | `@CF` | 1559212036 |
| VPPAdminTools | `@VPPAdminTools` | 1828439124 |

## Instalação

Use `config/Install_Mods.bat` na instalação Steam para copiar mods do Workshop para o servidor.

## Uso previsto desta pasta

- Lista de mods (`mod-list.txt`)
- Notas de compatibilidade entre versões
- Configurações específicas de mod (quando aplicável)
