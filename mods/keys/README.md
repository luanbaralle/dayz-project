# Chaves de mods (`.bikey`)

Coloque aqui chaves **customizadas** que não vêm automaticamente do Workshop.

| Origem | Quem copia para `server/keys/` |
|--------|--------------------------------|
| Workshop | `install_mods.sh` (automático) |
| Custom / manual | `deploy.sh` (via rsync desta pasta) |

**Não versione** chaves de mods que o Workshop já instala — `install_mods.sh` cuida disso.
