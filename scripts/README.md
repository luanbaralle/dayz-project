# Scripts utilitários

Scripts de manutenção e automação do projeto (não confundir com scripts de deploy em `deploy/`).

## Uso previsto

- Backup de perfis
- Validação de configuração
- Sincronização manual de arquivos
- Utilitários de diagnóstico

## Convenções

- Scripts Windows: extensão `.bat` ou `.ps1`
- Scripts Linux: extensão `.sh`
- Documente cada script com comentário no topo descrevendo uso e parâmetros
